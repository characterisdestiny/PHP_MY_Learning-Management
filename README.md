

# 나의 학습 관리 웹 (StudyHub)

## 프로젝트 배경
- 할 일(To-do)과 강의/스터디 일정을 한 번에 관리하기 위한 개인용 학습 관리 웹입니다.
- 복잡한 프레임워크 없이 PHP + MySQL만으로 가볍게 동작하도록 구현했습니다.
- 할 일의 우선순위·카테고리·마감일과 강의의 일정·진행도 등을 한 화면에서 정리해서
  학습 계획을 체계적으로 관리하는 것이 목표입니다.
- 메인 화면에서 라이트/다크 모드를 전환할 수 있어, 사용 환경에 맞는 테마로 학습할 수 있습니다.

## 폴더 구조

```
studyhub
├─ index.php          # 메인 대시보드: To-do / 강의 관리 진입, 라이트/다크 모드 전환
├─ todos.php          # To-do 리스트 관리 (추가, 완료 토글, 삭제, 우선순위·카테고리·마감일, 페이지네이션)
├─ lectures.php       # 강의/스터디 관리 (제목, 과목, 강사, 일정, 진행도, 검색·필터, 페이지네이션)
└─ includes
   └─ db.php          # MySQL 연결 설정 (localhost:3307, root, studyhub DB)
```

주요 기능 요약

대시보드 (index.php)

“To-do 리스트”, “강의/스터디 관리” 페이지로 이동

LocalStorage를 이용한 라이트/다크 모드 전환 버튼

To-do 관리 (todos.php)

할 일 추가: task, due_date, priority(높음/보통/낮음), category 입력

완료 상태 토글 및 삭제

최신 순/마감일 기준 목록 정렬, 페이지네이션

강의/스터디 관리 (lectures.php)

강의 등록: heading(제목), subject(과목), instructor(강사), schedule(일정), progress(진행도)

제목/과목/강사 키워드 검색, 날짜별 필터

목록 페이지네이션 및 삭제 기능
